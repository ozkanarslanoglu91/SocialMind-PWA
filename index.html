SocialMind

Modern siyah tema — yönetim paneli ve sosyal entegrasyonlar için temiz bir başlangıç.

Hızlı Başlangıç

Projeyi yerel olarak başlatmak için aşağıdaki komutları kullanın.

npm run dev
npm run build
dotnet run --project SocialMind/SocialMind.Web
Canlı Önizleme

Notlar

  • .NET servisleri SocialMind.Web içinde çalışır.
  • Swagger geliştirme ortamında tekrar etkinleştirilebilir.
  • Veritabanı ve kimlik bilgilerini `appsettings.secrets.json` ile yükleyin.
